🧾 Analysis: Focal Tversky Loss


1. Overall Performance
The model trained with Focal Tversky Loss achieved:
Epoch 1: F1=0.2707, IoU=0.2401
Epoch 2: F1=0.2908, IoU=0.2587 (peak performance)
Epoch 3: F1=0.2343, IoU=0.2038

The performance shows minor improvement in early epochs, followed by degradation, indicating unstable convergence.


2. Training Dynamics
The training behavior is characterized by:
Initial improvement in early training
No sustained learning across epochs
Degradation in later epochs

This suggests that the focal scaling does not stabilize optimization in this setting.


3. Class-wise Behavior
Per-class accuracy reveals:
Class 14: Acc=1.0000
All other classes: Acc=0.0000

This indicates that the model converges to a single dominant-class prediction, assigning all pixels to the most frequent class.


4. Effect of Focal Tversky Loss
Focal Tversky Loss is defined as:
Loss=(1-Tversky)^γ

It combines:
Tversky Loss → controls FP and FN trade-off
Focal scaling → emphasizes hard-to-classify samples


5. Observed Behavior
Despite this combination, the model exhibits:
No improvement in minority class learning
Continued dominance of a single class
Lack of meaningful multi-class representation

This indicates that:
Focal scaling does not improve class balance when overlap-based objectives already fail.


6. Failure Mode
The primary failure remains:
Collapse to a trivial dominant-class solution

Once the model converges to this state:
Most pixels are predicted correctly for the dominant class
Gradients for minority classes become negligible
Focal modulation becomes ineffective


7. Comparison with Tversky Loss
Property	        Tversky Loss	    Focal Tversky Loss
Stability	        Relatively stable	Less stable
Metrics	            Higher	            Lower
Class learning	    None	            None
Outcome	            Dominant class	    Dominant class

Focal Tversky does not provide improvement over standard Tversky Loss.


8. Key Insight
Focal Tversky Loss fails to recover from dominant-class collapse, as focal weighting becomes ineffective once the model converges to a trivial prediction strategy.


9. Interpretation
The results suggest:
Focal mechanisms require meaningful multi-class learning to be effective.

When the model collapses early:
No hard examples remain
Focal scaling has no corrective effect


10. Conclusion
Focal Tversky Loss:
Does not improve class imbalance handling
Does not prevent collapse
Performs worse than Tversky Loss
Fails to learn minority classes

Thus:
Focal Tversky Loss is ineffective for highly imbalanced multi-class segmentation tasks.
